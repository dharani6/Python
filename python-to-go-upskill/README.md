# Python to Go Upskill

This folder contains practice material for learning Python and Go together.
It is organized as a small hands-on training project with examples and notes.

## Contents

- `Day1a/`
  - `code/`
    - `list.py` - Python list example.
    - `Slice.go` - Go slice example.
    - `string.go` - Go string reversal example.
    - `string.py` - Python string reversal example.
  - `slides.pdf.pdf` - Presentation or notes for the first day.

## Purpose

The folder is designed for learners moving from Python to Go.
It includes simple code examples that demonstrate common operations in both languages.

## How to use

### Python examples

1. Open a terminal in `python-to-go-upskill/Day1a/code`
2. Run Python files with:
   ```powershell
   python string.py
   python list.py
   ```

### Go examples

1. Open a terminal in `python-to-go-upskill/Day1a/code`
2. Run Go files with:
   ```powershell
   go run string.go
   go run Slice.go
   ```

## Notes

- `string.go` and `string.py` show how to reverse a string in Go and Python.
- `Slice.go` shows a simple Go slice example.
- `list.py` shows a simple Python list example.
