# Python to Go Upskill

This folder contains practice examples for learning Python and Go side by side.
The goal is to help fresh learners understand common data structures and
operations in both languages, with simple code and clear documentation.

## Contents

- `Day1a/`
  - `code/`
    - `dictionary.py` - Python dictionary reversal example.
    - `fixed_array.go` - Go fixed-size array reversal example.
    - `Integer-Reversal.go` - Go integer reversal example (if present).
    - `list.py` - Python list reversal example.
    - `Map.go` - Go slice reversal example with input/output display.
    - `number.py` - Python number example (if present).
    - `Slice.go` - Go slice example.
    - `Stack-and-Queue-Reversal.go` - Go reversal example with stack/queue insight.
    - `Stack-and-Queue-Reversal.py` - Python stack and queue reversal examples.
    - `string.go` - Go string reversal example.
    - `string.py` - Python string reversal example.
    - `tuple.py` - Python tuple reversal example.
  - `slides.pdf` - Presentation or notes for the first day.

## Purpose

This folder is designed for learners moving from Python to Go.
It includes small practice programs that explain how to perform the same idea
in both languages, so learners can compare syntax and behavior.

## How to use

### Python examples

1. Open a terminal in `python-to-go-upskill/Day1a/code`
2. Run Python files with:
   ```powershell
   python list.py
   python string.py
   python tuple.py
   python dictionary.py
   python Stack-and-Queue-Reversal.py
   ```

### Go examples

1. Open a terminal in `python-to-go-upskill/Day1a/code`
2. Run Go files with:
   ```powershell
   go run string.go
   go run Slice.go
   go run fixed_array.go
   go run Map.go
   go run Stack-and-Queue-Reversal.go
   ```

## Notes

- Most examples include comments, input/output examples, and complexity notes.
- Python examples use slicing and built-in functions for simple reversal operations.
- Go examples use slices, arrays, and loops to show how to do the same tasks.
- The stack and queue examples explain LIFO and FIFO behavior clearly.

## Learning tips

- Read the comments at the top of each file first.
- Try running the examples and then modify the input values.
- Compare Python and Go versions to see how the same idea is expressed differently.
- Use the complexity notes to understand `O(n)` versus `O(1)` behavior.
